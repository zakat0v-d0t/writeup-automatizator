# Writeup Automator

Удобная CLI-утилита для автоматической генерации красивых и структурированных райтапов (writeups) после решения CTF-тасков.

## Требования 

- Python 3.13+

## Установка 

```
# 1. Клонируйте репозиторий  
git clone https://github.com/zakat0v-d0t/writeup-automatizator.git  
cd writeup-automator  
  
# 2. Установите зависимости и саму утилиту  
uv pip install -e .
```

## Использование

После установки утилита будет доступна глобально в вашем виртуальном окружении под именем `writeup`.

### Создание нового райтапа

Запустите команду инициализации:

```
writeup init
```

**Процесс:**

1. Утилита запросит базовую информацию: название таска, категорию (pwn, web, crypto и т.д.), сложность и флаг.

2. Далее начнется сбор шагов решения. Для каждого шага можно указать заголовок, текстовое описание и использованные терминальные команды (поддерживается перенос строк через `\\n`).

3. Чтобы завершить ввод шагов, просто оставьте заголовок очередного шага пустым и нажмите `Enter`.

4. В консоль будет выведен путь к сохраненному `.md` файлу.

### Просмотр доступных команд

```
writeup --help
```
